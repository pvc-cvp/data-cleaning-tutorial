# data-cleaning-tutorial
# 🧹 数据清洗入门教程

一个交互式数据清洗学习工具，通过 6 步可视化教程和 Python 实操沙盒，帮助初学者掌握数据清洗的核心方法。

## 🔗 在线体验

👉 **https://pvc-cvp.github.io/data-cleaning-tutorial/**

## 📖 内容概览

- **Step 01** — 总览：识别 5 类常见数据问题
- **Step 02** — 缺失值：填补空白数据
- **Step 03** — 去重：删除重复记录
- **Step 04** — 格式：统一数据表示方式
- **Step 05** — 异常值：修正不合理数值
- **Step 06** — 验证：检查格式合法性
- **🐍 Python 实操** — 在模拟沙盒中编写 pandas 代码完成清洗任务

## 🛠️ 技术栈

React + Vite + Tailwind CSS + Framer Motion
